#List of servers
#
#Add a server (without adding the whole Region:
#s\[name];[url];[Timezone]
#with [name] as the name displayed by Workbench
#with [url] the base server url
#with [Timezone] the JAVA name of the timezone the server is located in (Available options are shown on boot when in debugging mode)
#e.g.:
#s\zz1;https://zz1.tribalwars.works;Europe/London
#
#Adding a whole region:
#\[url];[Timezone]
